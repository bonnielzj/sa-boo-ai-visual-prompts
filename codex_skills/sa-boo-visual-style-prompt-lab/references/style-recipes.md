# Style Recipes for Immediate Prompt Variants

Combine one recipe with the prompt operating system. Keep the subject constant; change visual mechanism.

## SA&BOO house styles

### Liquid Silver Editorial

Chinese:
`{主体}，液态银色金属美学，水银般流体镀铬表面，冷白与淡蓝紫高光，极简高级编辑构图，大面积留白，柔和弥散轮廓光，细腻反射，奢侈品KV质感，高清，避免廉价塑料感和杂乱背景。`

English keywords:
`liquid silver metal, mercury chrome, iridescent highlights, luxury editorial, clean negative space, soft rim light, premium product visual`

### Technical Blueprint Luxury

Chinese:
`{主体}的高级产品设计蓝图，白底黑色精细线稿，正视图/侧视图/三分之四视图，结构拆解、尺寸线、材质注释，线条清晰可转矢量，商业印刷级，无阴影无杂乱。`

English keywords:
`technical blueprint, refined black line art, exploded view, material annotations, measurement lines, vector-ready, crisp white background`

### Victorian Ornamental System

Chinese:
`{主体}装饰线稿，维多利亚花卉雕版、洛可可卷草、文艺复兴藤蔓结构，精致极繁但秩序清晰，对称徽章/边框构图，白底黑线，可用于奢侈品包装与品牌辅助图形。`

English keywords:
`Victorian ornamental engraving, Rococo scrollwork, Renaissance vines, maximal floral line art, luxury emblem, vector-ready`

### East-West Understated Luxury

Chinese:
`{主体}，东方留白与西方现代编辑网格融合，低饱和米白/墨黑/银灰色系，天然材质与微弱金属光泽，安静、克制、有仪式感，空间叙事般的构图。`

English keywords:
`East-West luxury, negative space, restrained palette, natural material, quiet ceremony, editorial grid, symbolic minimalism`

## Canva/Adobe/Figma trend recipes

### Reality Warp / Surreal Editorial

- Mechanism: real + uncanny synthetic world.
- Use for: campaign concept, fashion/beauty, surreal product KV.
- Prompt add-on: `liminal atmosphere, impossible reflection, dreamlike distortion, editorial realism, slightly uncanny but elegant`.

### Texture Check / Multisensory Material

- Mechanism: surface-first tactile realism.
- Use for: product mockup, interior mood, brand material language.
- Prompt add-on: `macro tactile texture, glassy translucent surface, waxy softness, visible material grain, touchable depth, sensory detail`.

### Opt-Out / Quiet Editorial Authority

- Mechanism: less noise, serif, strict layout.
- Use for: high-end personal IP cover, proposal title page, luxury brand.
- Prompt add-on: `structured editorial grid, restrained palette, refined serif typography zone, clean logo area, calm authority`.

### Notes App Chic / Personal Authenticity

- Mechanism: candid fragments arranged with taste.
- Use for: Bonnie personal IP, behind-the-scenes, informal social content.
- Prompt add-on: `camera roll collage, handwritten note fragments, torn paper edges, casual marks, charming imperfection, authentic personal archive`.

### Drama Club / Cinematic Hook

- Mechanism: stage, spotlight, emotional climax.
- Use for: poster, launch cover, high-impact deck opener.
- Prompt add-on: `dramatic spotlight, velvet curtain shadow, cinematic still, bold title-safe composition, high contrast, theatrical mood`.

### Prompt Playground / Retro-Tech UI

- Mechanism: UI fragments become graphic material.
- Use for: AI tool content, design education, tech-personal IP.
- Prompt add-on: `retro computer interface, spreadsheet grid, dropdown menus, code snippets as abstract shapes, lo-fi UI nostalgia, playful data visualization`.

### Zinegeist / Controlled Anti-Gloss

- Mechanism: DIY paper + type rebellion.
- Use for: provocative opinion post, youth/experimental campaign.
- Prompt add-on: `photocopy grain, torn paper collage, oversized brutal typography zone, black ink, anti-gloss zine layout, expressive imperfection`.

### 3D Immersive Web Hero

- Mechanism: depth + interactivity implied in a still.
- Use for: landing page hero, Figma screen, product/service visualization.
- Prompt add-on: `3D immersive hero section, floating interface layers, WebGL-like depth, scroll-triggered composition, interactive product model, clean UI surface`.

### Dark Chrome Retrofuture

- Mechanism: dark UI + chrome/neon nostalgia.
- Use for: AI/fashion/beauty/music brands.
- Prompt add-on: `dark mode interface, chrome texture, neon edge light, retrofuturistic sci-fi gradient, pixel-era nostalgia refined into luxury`.

## Fast 6-style matrix

When user wants quick variants without specifying number, produce:

1. Liquid Silver Editorial
2. Texture Check Multisensory
3. Opt-Out Quiet Editorial
4. Reality Warp Surreal
5. Victorian Ornamental / East-West Luxury
6. Dark Chrome Retrofuture or Notes App Chic depending on audience
