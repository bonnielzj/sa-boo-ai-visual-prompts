# 07｜视觉资产链表与向量索引

为避免外部 URL 失效导致资料库失效，本仓库增加了本地视觉资产链表系统。

## 目录入口

- 本地视觉链表：`visual_assets/visual_asset_linked_list.md`
- 资产图谱：`visual_assets/asset_graph.json`
- 资产表：`visual_assets/visual_asset_manifest.csv`
- 缩略图缓存：`visual_assets/cache/thumbs/`
- 文本向量索引：`vector_index/visual_asset_vectors.json`
- 搜索脚本：`scripts/search_visual_assets.py`

## 使用方式

### 1. 直接浏览视觉链表

打开：

```text
visual_assets/visual_asset_linked_list.md
```

每个节点都有：

- 本地缩略图
- 原始来源页
- 原资产链接
- SHA256 / pHash proxy
- 视觉机制
- SA&BOO 转译
- prompt 标签
- 上下游链路

### 2. 用关键词搜索视觉资产

```bash
python3 scripts/search_visual_assets.py "Drama Club 戏剧 聚光灯 小红书封面"
```

### 3. 根据链路找到相近风格

在 `asset_graph.json` 里看：

- `semantic_related`：语义相似资产
- `next_in_platform` / `prev_in_platform`：同平台资产链表
- `prompt_recipe_reference`：关联到可直接复制的 Prompt 配方
