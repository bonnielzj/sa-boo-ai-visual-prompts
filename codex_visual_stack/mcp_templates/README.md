# MCP / Key / URL Templates

This folder intentionally contains **placeholders only**.

- Replace `${OPENAI_API_KEY}`, `${GITHUB_TOKEN}`, `${FIGMA_ACCESS_TOKEN}`, `${AIMAMI_RELAY_URL}`, `${AIMAMI_RELAY_TOKEN}` locally.
- Do not commit real keys, relay URLs, private account snapshots, `auth.json`, or `secrets.json`.
- SketchUp MCP is usually a local plugin/bridge; keep local absolute paths in your private config.
- AutoCAD/cadq paths vary by machine; use the template as a wiring map, not as a live config.
