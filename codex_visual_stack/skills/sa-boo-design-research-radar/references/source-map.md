# Source Map for SA&BOO Design Research

Use this when selecting sources for design trend research. For current facts, browse and cite the live page.

## Source hierarchy

1. **Primary / official**: platform reports, product docs, professional associations, annual trend reports.
2. **Industry media / communities**: ArchDaily, Dezeen, FRAME, Design Milk, Behance, LogoLounge, 优设, 站酷, 数英, TOPYS.
3. **Social/search signal**: 小红书、抖音、TikTok、Instagram、Pinterest. Treat as directional unless backed by data.
4. **Third-party data platforms**: 千瓜、新榜、巨量算数/巨量引擎、QuestMobile, Hootsuite, Sprout Social.
5. **User-provided evidence**: screenshots, exported notes, client chats, analytics, CRM keywords.

## Interior / renovation sources

- ASID research: wellness, sustainability, neuro-inclusive design, economic outlook.
- Houzz / Houzz Pro: home design predictions, renovation studies, homeowner search behavior.
- NKBA / KBIS: kitchen and bath trends, lighting, storage, smart tech, wellness.
- RIBA Plan of Work / AIA contract resources: stages, client communication, changes.
- Autodesk State of Design & Make / Forma / Revit: AI, BIM, early-stage analysis, design data.
- ArchDaily / Dezeen / FRAME / Designboom: global material, hospitality, retail, workplace, and residential signals.
- 酷家乐、Coohom、D5 Render、Twinmotion、Enscape/Chaos: production and visualization tools.
- 中国家居家装 reports: 树懒生活Fine、行业协会、家居媒体、巨量/小红书行业月报.

## Brand / visual identity sources

- Adobe Creative Trends / Adobe Stock: visual style, AI, multi-sensory, authenticity.
- Canva Design Trends: creator behavior, social design formats, accessible visual trends.
- Figma Blog / release notes / Config: design-to-code, Buzz, Make, Sites, Draw, MCP/Code Connect.
- LogoLounge: logo/identity patterns and symbol evolution.
- Behance trends / Adobe Portfolio: portfolio/case structure and visual execution.
- Pinterest Predicts / TikTok What's Next: consumer aesthetics and social behavior.
- 优设 / 站酷 / 数英 / TOPYS: Chinese designer learning, AIGC workflows, brand campaigns.

## Social platform observation

For 小红书 / 抖音:
- Track keyword clusters: 装修避坑, 旧房改造, 软装设计, 别墅装修, 全屋定制, 奶油风, 原木风, 法式, 显贵配色, 设计师审美, 品牌升级.
- Record: search autocomplete, top note titles, cover patterns, comment questions, pain points, CTA formats.
- Use 千瓜/新榜/蒲公英/聚光/巨量算数/巨量引擎 when available for quantitative support.
- Do not scrape private/login-gated content without permission. Ask user for screenshots/exports when needed.
