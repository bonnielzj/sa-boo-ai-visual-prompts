# SA&BOO Trend Taxonomy

Use this to convert raw signals into SA&BOO-relevant opportunities.

## Core filters

A trend is worth SA&BOO attention if it supports at least two of:

- **Aesthetic fit**: 高级、克制、先锋、材质转译、东方秩序、低调奢华、文化深度.
- **Commercial fit**: improves client trust, higher fee justification, faster conversion, repeatable package.
- **Workflow fit**: speeds research, concept, visualization, documentation, procurement, content production.
- **Content fit**: can become a strong Bonnie POV or teachable social post.
- **Durability**: likely to last beyond a seasonal style label.

## Categories

### Interior / renovation

1. 存量房 / 局改 / 轻全案
2. 厨卫升级 / 收纳 / 五金 / 灯光
3. 健康、适老、神经友好、声光热空气
4. 东方园林秩序 + 西方古典比例 + 当代材料
5. 手工、复古、个人叙事、慢装饰
6. 可持续 / 低VOC / 耐久 / 可维护材料
7. BIM/CAD/3D/实时渲染/云设计协同
8. 标准化交付、变更、采购、复盘

### Brand / visual identity

1. 动态品牌系统 / motion identity
2. 材质化、3D、多感官视觉
3. 反AI光滑感：手作、纹理、瑕疵、真实过程
4. 结构化剪贴簿 / zine / local flavors
5. 液态银、chrome、Y2K/cyber luxury
6. 极简与极繁并置、复古未来主义、奇异历史主义
7. Figma as brand production system
8. Portfolio as sales case: problem → strategy → system → result

### AI / model workflow

1. Fast concept: GPT Image / Midjourney / Firefly
2. Commercial-safe brand production: Firefly / licensed assets / in-house references
3. Controlled generation: ControlNet, IP-Adapter, reference images, masking, inpainting
4. Style memory: style reference, custom model, LoRA-like fine-tuning where legally safe
5. Video/motion: Runway, CapCut/剪映, Adobe, Figma prototypes
6. Layout/text-heavy graphics: prefer models/tools with stronger typography and editing control

## Scoring table

Score 1-5 for each:

- Evidence strength
- SA&BOO aesthetic fit
- Client conversion value
- Operational leverage
- Learning difficulty (reverse: 5 = easy)
- Risk level (reverse: 5 = low risk)

Recommend:
- 24+ = act this month
- 18-23 = test in pilot
- 12-17 = monitor
- below 12 = ignore unless client asks
