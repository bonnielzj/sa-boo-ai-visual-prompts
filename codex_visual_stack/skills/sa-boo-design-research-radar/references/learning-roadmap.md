# SA&BOO Learning Roadmap

Use this to produce a learning plan. Adjust to current project needs.

## 30-day priority: conversion and speed

### Interior
- AutoCAD drawing standards: layers, dimensions, title blocks, revision marks, export naming.
- SketchUp/Rhino quick massing and space tests for client communication.
- D5 / Enscape / Twinmotion real-time workflow: camera, lighting, material library, before/after views.
- Lighting basics: natural/task/accent lighting, color temperature, glare, kitchen/bath task lighting.
- Material board discipline: stone, wood, metal, textile, low-VOC claims, maintenance, cost tier.

### Brand / content
- Figma brand kit: variables, styles, components, social cover templates, proposal templates.
- Motion basics: logo reveal, social intro/outro, Lottie/Rive/AE simple loops.
- Case study writing: problem, insight, concept, system, application, results.
- Xiaohongshu/Douyin search content: keyword + POV + visual proof + CTA.

### AI
- Prompt systems for SA&BOO styles: East-West luxury, liquid silver, blueprint line art, Victorian ornament, Memphis/color.
- GPT Image / Midjourney / Firefly model selection and output QA.
- Inpainting/masking/editing for controlled iteration.
- Privacy workflow: anonymize client plans/photos before upload.

## 90-day priority: defensible workflow

### Interior
- Revit/BIM basics for complex projects: families, schedules, sheets, coordination.
- Forma / early analysis mindset: site, sun, wind, massing, carbon story when relevant.
- Procurement and specification database: supplier, lead time, dimensions, finish, warranty, price band.
- Change-order SOP: decision log, signed confirmation, version control.

### Brand
- Figma Buzz/Sites/Make exploration for rapid brand pages and marketing templates.
- 3D brand visuals: Blender/Spline/Substance basics for tactile brand systems.
- Dynamic identity rules: motion grammar, responsive logo, social crops, event/space signage.

### AI/model algorithms
- ControlNet: structure control from edges, depth, pose, or sketches.
- IP-Adapter / image prompt: keep visual identity from reference images without copying exact content.
- LoRA/custom models: understand concept, legal/data requirements, and when not to train.
- ComfyUI: node-based reproducible workflow for advanced control.

## 180-day priority: SA&BOO moat

- Internal trend radar database updated monthly.
- SA&BOO material + color + prompt library.
- Repeatable concept board generator: brief → three directions → prompts → risk check.
- Sales-type portfolio system for Behance/website/小红书合集.
- Project-after-content pipeline: every project becomes case study, short video, carousel, and learning post.

## Practice tasks

- Recreate one high-end interior mood in 3 tools: AI concept, SketchUp/D5, Figma proposal page.
- Turn one client pain point into 5 小红书 notes: 搜索型, 避坑型, 审美教育型, 幕后型, 转化型.
- Build one brand identity mini-system: logo logic, palette, type, texture, motion, 9 application mockups.
- Create one controlled AI workflow: reference image + prompt + mask + final QA, with before/after notes.
