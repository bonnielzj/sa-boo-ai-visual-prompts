# SA&BOO Visual-First Rule Reference

Use this reference when a SA&BOO task needs a stricter explanation of the visual-first doctrine.

## Principle

视觉链表为主，文字为辅。Visual assets are the evidence and taste carrier; text is the interpretation and decision system.

## Reject list

Reject assets that are blurry, unreadable, over-compressed, untraceable, aesthetically weak, irrelevant, or legally unsafe for the intended use.

## Minimum output for preserved assets

`asset_id`, `thumbnail_path`, `local_original_path`, `source_url`, `rights_status`, `quality_notes`, `prev_id`, `next_id`, `tags`.

## Efficient exception

For quick non-visual tasks, do not force visual decoration. State that no visual asset is required and keep the output concise.
