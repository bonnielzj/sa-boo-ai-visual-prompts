# QA / Risk Checklist

## Material data

- [ ] Category and subcategory are filled.
- [ ] Unit is correct and consistent.
- [ ] Size/spec includes thickness, finish, color, model, and installation condition where needed.
- [ ] Source URL/catalog/quote is attached.
- [ ] Last verified date is present.
- [ ] Sustainability/low-VOC/fire/waterproof claims have proof.

## Supplier data

- [ ] Contact is current.
- [ ] Payment terms are recorded.
- [ ] Lead time and MOQ are recorded.
- [ ] Warranty/after-sales info recorded.
- [ ] Supplier score updated after project completion.

## Budget data

- [ ] Quantity × unit price formulas correct.
- [ ] Tax, freight, installation included or explicitly excluded.
- [ ] Quote validity date checked.
- [ ] Approved amount differs from estimate only with explanation.
- [ ] Variance and contingency visible.

## Procurement

- [ ] Required date aligns with project schedule.
- [ ] Long-lead/high-risk items flagged early.
- [ ] Client approval evidence saved.
- [ ] Substitutions logged with cost/schedule/design impact.
- [ ] Delivered/installed status verified.

## Client privacy

- [ ] Internal discounts/margins hidden before sharing.
- [ ] Supplier private contacts hidden before sharing.
- [ ] Client address/budget protected in external files.
