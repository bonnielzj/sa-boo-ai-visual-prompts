# Before / After Standard

Before/after is not only visual drama; it is a sales argument. It should show the value of design decisions.

## Pairing rules

- Use the same camera, crop, focal length, exposure, time of day, and aspect ratio.
- Compare one clear design improvement per pair when possible.
- Label before/after clearly but quietly.
- Do not make the before artificially ugly beyond reality.
- Keep before images honest; if based on site photo/model reconstruction, state it.

## Pair types

| Pair | Use |
|---|---|
| Existing → Proposed | renovation transformation |
| Option A → Option B | material/color decision |
| Day → Evening | lighting value |
| Empty shell → Furnished | soft furnishing value |
| Plan → Render | helps client understand drawing-to-space translation |

## Standard comparison page

```text
Title: Before / After — 客厅光线与材质升级
Left: Before image
Right: After image
Bottom: 3 bullets explaining change
1. 动线：...
2. 光线：...
3. 材质：...
```

## Social/case-study copy formula

```text
不是把空间“变豪华”，而是把{问题}转化为{体验}。
Before：{原始痛点}
After：{设计动作}
价值：{生活/审美/商业结果}
```

## QA

- [ ] Camera locked.
- [ ] Lighting matched unless the comparison is explicitly day/evening.
- [ ] Labels readable.
- [ ] Transformation is truthful.
- [ ] Client privacy checked.
