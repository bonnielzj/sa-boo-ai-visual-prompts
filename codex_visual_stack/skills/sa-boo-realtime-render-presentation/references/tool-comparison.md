# D5 / Enscape / Twinmotion Tool Comparison

## Quick decision

| Need | Best tool | Why |
|---|---|---|
| Fast high-end interior stills | D5 Render | Strong real-time visual quality, interior asset/material workflow, cinematic image output |
| Live SketchUp/Revit/Rhino design review | Enscape | Stays close to design model; easy walkthroughs and quick visual settings |
| Landscape / villa / large context / phasing | Twinmotion | Strong context, vegetation, weather, phasing, media/presentation tools |
| Client meeting while modeling | Enscape | Low friction and direct model feedback |
| Final mood images for proposal | D5 or Twinmotion | Stronger storytelling and media staging |
| Construction sequence / before-after / option states | Twinmotion or D5 | Use scene states/configurations/phasing where available |

## SA&BOO default stack

1. SketchUp/Rhino/Revit model for geometry.
2. D5 for fast high-end interior stills and short videos.
3. Enscape for live design decisions and client walkthroughs.
4. Twinmotion for villa/exterior/context and phasing-heavy stories.
5. Figma/PDF for final presentation packaging.

## Tool-specific strengths

### D5 Render

Use for:
- Interior mood stills.
- Material-rich closeups.
- Fast asset population.
- Atmospheric client images.
- Short cinematic walkthroughs.

Watch:
- Do not overuse stock assets.
- Control exposure and reflections; avoid plasticky over-gloss.
- Keep camera names and scene states organized.

### Enscape

Use for:
- Working-session walkthroughs.
- Quick “designer + client” co-review.
- Revit/SketchUp/Rhino synchronization.
- Rapid daylight and material checks.

Watch:
- It can reveal model sloppiness; clean geometry before review.
- Default visual settings can look generic; create SA&BOO presets.
- Avoid presenting unapproved WIP as final photoreal renders.

### Twinmotion

Use for:
- Villas, gardens, exterior/context scenes.
- Before/after and phasing stories.
- Media sets: images, videos, panoramas, presentations.
- Seasonal/weather/day-night atmosphere.

Watch:
- Large asset scenes can become heavy.
- Too many animated people/plants can distract from design.
- Keep material substitution consistent with real specs.
