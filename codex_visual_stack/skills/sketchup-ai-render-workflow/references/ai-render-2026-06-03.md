## Scene-prep checklist for interiors

- Units/model scale correct.
- Camera is named and saved as a SketchUp Scene.
- Geometry is simplified but spatially accurate.
- Major planes assigned simple color/material tags: wall, floor, ceiling, built-in, furniture, metal, glass, textile, greenery.
- Use white/off-white base material for massing passes, then prompt material detail.
- Add minimal entourage only when necessary; do not let decorative clutter hide layout logic.
- Set shadow/time-of-day intentionally if the viewport is used as image guidance.

## Prompt formula

Use this structure:

```text
[room type + camera], [spatial intent], [style direction],
[preserved geometry instruction],
[floor material], [wall material], [ceiling treatment], [built-in details], [furniture details],
[lighting story], [mood],
photorealistic interior rendering, natural proportions, high-end but buildable, clean composition
```

Negative prompt formula:

```text
distorted geometry, changed layout, extra doors, extra windows, warped ceiling, wrong scale, messy clutter, people, text, watermark, logo, cartoon, overexposed, oversaturated, plastic materials, impossible lighting
```

## Influence guidance

- White model to mood: geometry influence 55–70%, prompt influence 65–85%.
- Fixed-layout material test: geometry influence 75–90%, prompt influence 55–75%.
- Client-facing option comparison: geometry influence 85–95%, prompt influence 45–65%.
- Decorative/inpainting change: mask only the target area; use high geometry preservation language.

Actual slider names or ranges may vary by plugin build; translate these values into the closest available controls.

## Output naming

Recommended file naming:

```text
项目名_AIrender_空间_镜头_方向_YYYYMMDD_v01.png
项目名_AIrender_空间_镜头_prompt_YYYYMMDD_v01.txt
```

Keep a text file beside every saved render containing:

- SketchUp file path and scene name.
- Prompt and negative prompt.
- Influence/slider values.
- Manual notes: what succeeded, what failed, what to regenerate.

## Privacy and client-risk notes

- Use anonymized or cropped model views for early cloud AI tests when possible.
- Avoid uploading confidential full-floor plans, addresses, family photos, unreleased brand assets, or client-specific artwork without approval.
- Use AI Render for concept/mood/material exploration; final design decisions still require dimensioned drawings, material samples, supplier data, and buildability checks.
