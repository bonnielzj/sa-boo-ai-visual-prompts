# SA&BOO Figma Brand Kit Operating Method

## Goal

Create a working brand production file, not only a guideline PDF. It should support:

- Interior proposal decks
- Brand proposal decks
- Xiaohongshu/Douyin covers and carousels
- Project case-study pages
- Moodboards and material boards
- Quote/invoice covers and client-facing documents
- Website/Behance case visuals

## V1 scope: build in this order

### 1. Foundations

- Color primitives: ivory, ink, warm gray, stone, champagne, bronze, silver, accent red.
- Semantic colors: backgrounds, surfaces, text, border, accent, overlay.
- Typography: display, title, subtitle, body, caption, quote, label.
- Spacing: 4, 8, 12, 16, 24, 32, 48, 64, 96.
- Radius: 0, 4, 8, 16, 999.
- Shadows/effects: soft paper, gallery card, chrome glow.
- Image rules: high-contrast crop, material close-up, negative space, editorial cover.

### 2. Brand assets

- Logo lockup: horizontal, stacked, symbol-only, monochrome, reversed.
- Safety area and minimum size.
- Symbol/motif library: liquid silver, Eastern garden line, Victorian ornament, blueprint linework, Memphis color block.
- Texture swatches: stone, silk, chrome, paper grain, shadow gradient.

### 3. Components

Build only the components Bonnie will repeatedly use:

- `Cover / Editorial`
- `Cover / Xiaohongshu`
- `Header / Proposal`
- `Section Divider`
- `Text Block / Concept`
- `Image Card / Material`
- `Moodboard Tile`
- `Case Study / BeforeAfter`
- `Quote Block`
- `CTA Footer`
- `Label Chip`
- `Spec Table` for FF&E / materials

### 4. Templates

- Proposal cover, 16:9 and A4.
- Concept page.
- Moodboard/material board.
- Case-study spread.
- Xiaohongshu cover 3:4.
- Xiaohongshu carousel page.
- Douyin vertical cover 9:16.
- Behance/website case hero.


## Implementation phases with Figma MCP

1. Inspect or create file.
2. Create variables and text/effect styles.
3. Create documentation frames for foundations.
4. Create one component family at a time.
5. Assemble templates from instances.
6. Export sample PNG/PDF for QA.

## Minimum viable Brand Kit in one work session

If time is limited, build only:

- 20-30 color variables
- 8 text styles
- 8 spacing variables
- 5 components
- 4 templates

This is enough to start producing consistent SA&BOO materials.
