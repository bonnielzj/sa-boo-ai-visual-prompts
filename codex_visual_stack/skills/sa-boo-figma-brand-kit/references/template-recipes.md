# Template Recipes

## Proposal cover

Size: A4 portrait and 16:9.  
Structure: logo → project type → project name → concept line → date/version → large editorial image or abstract motif.  
Use: client proposal, PDF cover.

## Concept page

Structure: left narrative block, right reference/mood image, bottom material chips.  
Fields: concept title, 80-120 Chinese words, 3 keywords, material palette.

## Moodboard/material board

Structure: 6-10 image slots, 6 material swatches, 3 color chips, short design logic.  
Use: interior and brand concepts.

## Case-study spread

Structure: problem → insight → visual/spatial motif → application → result.  
Use: Behance, website, PDF, Xiaohongshu carousel.

## Xiaohongshu cover

Size: 3:4, e.g. 1242×1660 or 1080×1440.  
Structure: strong title, one visual hero, small SA&BOO mark, high contrast.  
Title style: searchable + opinionated, not keyword stuffing.

## Douyin/Reels cover

Size: 9:16, e.g. 1080×1920.  
Structure: title safe area, face/object or spatial hero, subtitle, episode label.

## Brand application board

Structure: logo, color, typography, pattern, mockup, social/application tiles.  
Use: brand proposal and identity case presentation.
