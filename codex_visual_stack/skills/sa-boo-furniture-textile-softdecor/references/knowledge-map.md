# SA&BOO Furniture / Textile / Soft Decor Knowledge Map

## Purpose

Build a living knowledge base that connects furniture history, product intelligence, textile performance, visual references, local assets, open datasets, CAD blocks, SketchUp components, renders, FF&E schedules, and Bonnie's evolving aesthetic criteria.

## Learning cadence

1. **Weekly object family**: one family such as sofa, lounge chair, dining table, rug, curtain, bed, casegoods, decorative lighting.
2. **Collect 12-30 visuals**: mix official products, museum/historical examples, strong contemporary interiors, local project examples, open-license assets.
3. **Write 5-10 object cards**: one card per archetype, not per random image.
4. **Translate to CAD/SU**: define footprint, clearance, component tag, model LOD, material slots, render texture scale.
5. **Add FF&E line items**: supplier, finish, dimensions, price band, lead time, maintenance, certification.
6. **Write one aesthetic principle**: what SA&BOO should repeat or avoid.

## Learning priorities

### P0 — Must build first

- Sofa and lounge chair typologies: proportions, cushion construction, upholstery, comfort vs silhouette.
- Dining chair/table clearances and scale.
- Curtains and window treatments: fullness, track/pole, lining, acoustic/light effect.
- Rugs: size logic, pile, fiber, edge, placement under furniture.
- Bed/bedding: layered softness, hotel vs residential, headboard construction.
- Coffee/side tables: height relationships, stone/wood/metal/glass behavior.
- Decorative lighting: scale, glare, focal atmosphere, wiring/maintenance.

### P1 — Extend after P0

- Casegoods, consoles, shelves, wardrobes, loose storage.
- Art, mirrors, vessels, plants, books, objects, scent, seasonal styling.
- Outdoor furniture, performance textiles, hospitality/office FF&E.
- Custom furniture details for millwork coordination.

## Knowledge quality levels

- **Level 1: Named** — object has a label only; not enough.
- **Level 2: Visible** — has source image and basic tags.
- **Level 3: Useful** — has dimensions, material, style, use case, CAD/SU note.
- **Level 4: Specifiable** — has supplier, finish, budget, lead time, maintenance, code/performance notes.
- **Level 5: SA&BOO intelligence** — includes aesthetic judgment, local examples, rejection criteria, render/model implications, and reusable patterns.

Aim for Level 4-5 for recurring object families.
