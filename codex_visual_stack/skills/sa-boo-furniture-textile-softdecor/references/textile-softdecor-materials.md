# Textile, Upholstery, and Soft Decoration Materials

Use this for upholstery, curtains, rugs, bedding, cushions, wall textile, acoustic fabric, leather, and soft-styling choices.

## Textile decision stack

```text
use case → fiber → yarn/weave/knit → finish/backing → color/pattern → performance → maintenance → tactile/aesthetic role
```

Do not select textile only by color. Ask what it must do: sit on, filter light, absorb sound, soften architecture, add ritual, hide wear, meet code, resist pets/children, or create seasonal mood.

## Fiber behavior cheat sheet

| Fiber/material | Strength | Risk |
|---|---|---|
| Wool | resilient, warm, naturally soil-resistant, good rugs/upholstery | moths, cost, some cleaning sensitivity |
| Linen | elegant slub, relaxed light, strong dry fiber | wrinkles, abrasion/UV concerns, casual look |
| Cotton | breathable, familiar, printable | stains, shrinkage, lower abrasion unless blended/treated |
| Silk | luminous, luxurious, refined curtains/pillows | UV damage, delicate, expensive maintenance |
| Polyester | durable, colorfast, stain-resistant options, performance fabrics | can look flat/cheap if texture is poor |
| Acrylic/olefin | outdoor/performance uses, UV/mildew options | tactile quality varies |
| Viscose/rayon | soft sheen, silk-like rugs/textiles | moisture staining, pile crushing risk |
| Leather | patina, luxury, wipeability depending finish | scratches, heat/sun, ethical/cost concerns |
| Faux leather | cost/control, wipeable | peeling, poor breathability, shorter life if low grade |

## Upholstery performance checks

For contract or heavy-use settings, look for documented test data rather than marketing words:

- abrasion/double rubs or Martindale;
- pilling;
- seam slippage;
- colorfastness to light and crocking;
- flammability requirements for the market/project;
- cleaning code and bleach-cleanability if needed;
- antimicrobial/stain treatments only when meaningful and documented;
- VOC/emissions where indoor air quality matters.

Use ACT-style textile icons/guidelines as a reference pattern for abrasion, flammability, wet/dry crocking, colorfastness, and physical properties. Confirm the actual project code and supplier test report.

## Curtains / window treatment logic

Inspect:

- function: privacy, blackout, sheer light, acoustic, thermal, decorative framing;
- fullness: flat vs 1.5x/2x/2.5x+ gathering;
- heading: ripple fold, pinch pleat, wave, grommet, rod pocket;
- lining/interlining: blackout, thermal, body, luxury drape;
- track/pole: ceiling-recessed, exposed rod, motorized, stack-back space;
- floor relation: kiss, break, puddle; avoid accidental shortness;
- sunlight exposure: UV fading and fiber choice.

SA&BOO rule: curtains are architecture-softening devices. They must coordinate with ceiling, wall reveals, HVAC, lighting coves, and window frames, not be an afterthought.

## Rug logic

Inspect:

- fiber: wool, silk, viscose, jute, cotton, synthetic;
- construction: hand-knotted, tufted, woven, flatweave, machine-made;
- pile height and density;
- edge/binding/fringe;
- stain/cleaning requirements;
- furniture placement and rug pad.

Scale rule: a rug should anchor a furniture group. If it only sits under the coffee table without touching the seating composition, it often weakens the room.

## Bedding and cushions

Layer by function, not clutter:

- mattress/base and bed skirt/platform expression;
- fitted sheet/top sheet/duvet/coverlet/throw;
- sleeping pillows vs decorative pillows;
- cushion insert quality and karate-chop clichés to avoid;
- textile contrast: matte/gloss, rough/smooth, warm/cool, dense/airy.

## Soft decor hierarchy

1. **Architectural soft layer**: curtains, rugs, upholstered wall/headboard.
2. **Furniture textile layer**: upholstery and leather.
3. **Human-touch layer**: bedding, cushion, throw.
4. **Styling layer**: books, tray, vessel, art, plant, scent.

Avoid random small props before the large soft layers are correct.
