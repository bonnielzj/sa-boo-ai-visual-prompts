# Ergonomics / Proportion / Classical Design Basics

Use this when checking human comfort, accessibility, furniture dimensions, system furniture modules, CAD/SU scale, visual proportion, composition, or “classic design common sense.” This file is a **design QA layer**, not a substitute for project codes, brand technical sheets, field measurements, or local regulations.

## Core rule

```text
human body → activity → clearance/reach → object dimension → spatial proportion → CAD/SU verification → design judgment
```

A beautiful proportion that fails the body is not good design. A technically correct dimension that feels visually restless should be proportionally refined after ergonomic needs are protected.

## Source hierarchy

1. Local code / project standards / accessibility requirements.
2. Verified brand/product technical sheets.
3. Anthropometric and ergonomics guidance from reputable sources.
4. Classical proportion systems as composition tools.
5. SA&BOO aesthetic judgment and project-specific lifestyle rituals.

## Anthropometric method

- Define the user group: adult/child/elderly, residential/office/hospitality, region, accessibility needs.
- Decide whether the dimension is a **reach**, **clearance**, **support**, **view**, **circulation**, or **safety** dimension.
- For clearances, design for larger bodies and assistive devices where relevant.
- For reaches and controls, design for smaller/limited-reach users where relevant.
- Use adjustability when a single fixed dimension cannot fit the user range.
- Never copy a dimension without knowing whether it is for 5th percentile, 50th percentile, 95th percentile, or code minimum.

## Essential accessibility anchors

Use local accessibility standards for actual projects. For US ADA-based checks, store these as quick anchors:

| Item | Anchor | Use |
|---|---:|---|
| Clear floor / ground space | 30 in × 48 in / 760 × 1220 mm minimum | Forward or side approach to accessible elements |
| Turning circle | 60 in / 1525 mm diameter minimum | Wheelchair turning in required spaces |
| T-shaped turning | 60 in square with 36 in / 915 mm wide arms/base | Alternative turning geometry |
| Knee/toe clearance width | 30 in / 760 mm minimum | Dining/work surfaces, sinks, lavatories where required |
| Knee/toe depth | 17-25 in / 430-635 mm range depending condition | Pull-up approach and reach |

Treat these as **minimum compliance anchors**, not luxury comfort dimensions.

## Furniture ergonomic starter heuristics

These are starting points for design review; verify against product data, user needs, and project standards.

| Object | Common check | Design concern |
|---|---|---|
| Dining chair | Seat height vs dining table height | Thigh clearance and comfortable elbow/shoulder posture |
| Lounge chair / sofa | Seat height, depth, back angle, cushion firmness | Upright conversation vs lounging posture |
| Coffee table | Height relative to sofa seat | Reach, circulation, visual weight |
| Desk / work chair | Elbow height, monitor eye line, foot support | Avoid fixed furniture that forces posture problems |
| Bed / nightstand | Mattress top and nightstand height relation | Reach from lying/sitting, lamp/switch access |
| Wardrobe / closet | Hanging height, drawer reach, shoe/luggage zones | Daily ritual, visibility, heavy-lift risk |
| Kitchen | Worktop, appliance, sink/cooktop relation | Repetitive reach, heat/water safety, task sequence |
| Shelf / wall cabinet | Reach and load | Avoid high daily-use storage beyond comfortable reach |

## Proportion systems as design tools

### Golden ratio / phi

- Use `1 : 1.618` as a compositional guide for rectangles, panel divisions, art placement, layout rhythm, and negative-space relationships.
- Do **not** treat it as automatic beauty. It can organize attention, but poor function/material/detail will still fail.
- In interiors, often use softer approximations such as `60/40`, `5/8`, or Fibonacci-like steps rather than forcing exact phi.

### Rule of thirds

Use for quick composition in elevations, render cameras, moodboards, art placement, and proposal pages. It is faster than exact phi and often enough to prevent dead-center stiffness.

### Modulor / human-scale proportional systems

Use as a reminder that architectural proportion can be tied to human scale. It is useful for thinking about vertical datums, hand/eye reach, panel heights, guardrails, furniture and built-in relationships. Do not use it as a universal demographic standard.

### Classical orders / whole-number ratios

Classical architecture often uses readable proportional families and profiles. For interiors, learn from:

- base / middle / top hierarchy;
- column/panel rhythm;
- whole-number ratios such as 1:1, 1:2, 2:3, 3:4, 4:5, 4:9;
- profile depth, shadow, and hierarchy;
- repetition with controlled variation.

### Grids and modular systems

Use grids to connect CAD/SU/buildability with beauty:

- structural bay / wall panel module;
- cabinet module widths;
- tile/stone slab module;
- furniture grouping module;
- lighting grid and ceiling module;
- proposal page and moodboard grid.

## SA&BOO proportion review questions

1. Does the human body fit before the image looks good?
2. Is the main object hierarchy visible within 3 seconds?
3. Are big masses divided with a clear base/middle/top or field/frame logic?
4. Do panel seams align with architecture, openings, lighting, furniture, or module rules?
5. Does the negative space feel intentional rather than empty?
6. Are proportions tied to a ritual: sitting, cooking, dressing, reading, gathering, display?
7. Does the SketchUp model match verified dimensions or only “look about right”?
8. Can the CAD drawing dimension and annotate this design without lying?
9. Is the golden ratio/rule/grid helping the design, or being forced as decoration?
10. What dimension or proportion must be verified before this becomes client/施工图 truth?
