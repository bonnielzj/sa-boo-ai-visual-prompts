# SA&BOO Victorian Ornamental Line

## SA&BOO visual-first hard rule

- Treat `sa-boo-visual-first-core` as the governing rule for SA&BOO work: build or cite a high-quality visual asset chain before writing long text whenever the task involves design learning, research, style, proposal, prompt, rendering, CAD/FF&E assets, or project review.
- Use `sa-boo-visual-asset-index` when assets must be cached, cited, tagged, linked, or searched.
- Reject low-quality assets: blurry, unreadable, over-compressed, unattributed, aesthetically weak, irrelevant, or legally unsafe. **宁缺毋滥。**
- Keep the method efficient: do not force irrelevant images into quick text-only answers; for substantial outputs, include asset IDs, thumbnails/paths, sources, rights status, and quality notes.
- Let text serve the visuals: summarize, compare, decide, and translate; do not replace visual evidence with pure prose.

Create ornate, precise, classical decorative line prompts for luxury brands, packaging, certificates, emblems, and pattern systems.

## Style DNA

- Historical sources: Victorian ornament, Rococo scrolls, Renaissance intertwining vines, classical emblems, botanical engraving.
- Motifs: peony, orchid, rose, ribbon bird, butterflies, wings, stars, clouds, moon, scrollwork, gothic rose, cross, heart.
- Line quality: fine pen line, delicate, layered, rhythmic, clear, elegant, print-ready.
- Composition: symmetrical or asymmetric, central medallion, frame, border, crest, repeated pattern.
- Mood: elegant, ceremonial, collectible, high-end, decorative, narrative.

## Prompt recipe

1. Define main motif.
2. Select ornament system: Victorian / Rococo / Renaissance / Gothic / botanical engraving.
3. Set line language: black line on white, blue-white, gold foil line, silver line on black.
4. Add composition: frame, medallion, border, crest, pattern tile.
5. Add commercial use: packaging, certificate, logo auxiliary graphic, textile, stationery.

## Chinese prompt skeleton

```text
{主体}装饰线稿，融合维多利亚时代装饰、洛可可涡卷纹与文艺复兴绞缠纹，精致极繁主义，花卉/藤蔓/翅膀/徽章元素环绕构成。采用精细钢笔线描技法，线条流畅细腻，层次丰富，具有强烈装饰性、韵律感与高级品牌质感。构图{对称/不对称/圆形徽章/边框}，白底黑线，四周留白，商业印刷标准，可转矢量，高清，无阴影。
```

## English prompt skeleton

```text
ornamental line-art illustration of {subject}, Victorian decorative engraving, Rococo scrollwork, Renaissance intertwined vines, refined maximalist floral ornament, delicate pen lines, elegant layered details, {composition}, black ink on white background, luxury packaging / certificate / emblem aesthetic, vector-ready, crisp commercial print quality, high resolution, no shadow.
```

## Negative constraints

Avoid cartoon style, low-detail clipart, blurry linework, random text, messy clutter, fake watermark, distorted symmetry unless asymmetry is requested.
