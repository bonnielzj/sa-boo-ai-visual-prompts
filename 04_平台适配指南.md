# 平台适配指南｜Midjourney / DALL·E / Stable Diffusion / 即梦 / Runway

## 1. DALL·E / gpt-image / Codex imagegen

适合：商业概念图、产品KV、社交封面、室内概念、干净高级图。

写法：自然语言 + 明确约束。

示例：

```text
Create a premium social media cover visual of a liquid silver orchid, mercury-like chrome petals, soft diffused rim light, black-silver palette, centered editorial composition with large negative space on the upper third for Chinese title. Luxury, mysterious, refined. Avoid random text, watermark, cluttered background, cheap plastic look.
```

## 2. Midjourney

适合：Moodboard、艺术方向、氛围探索、时尚/品牌/空间概念。

写法：关键词密度更高，可加入画幅参数。

示例：

```text
liquid silver orchid, mercury chrome petals, luxury editorial still life, black silver palette, soft rim light, cinematic diffusion, clean negative space, mysterious futuristic mood, premium fashion art direction --ar 4:5 --stylize 250
```

## 3. Stable Diffusion / SDXL / Flux 类工具

适合：批量生成、固定风格、局部重绘、可控模型工作流。

Positive Prompt：

```text
liquid silver orchid, mercury chrome texture, luxury editorial composition, soft rim light, black-silver palette, clean negative space, high detail, refined reflections
```

Negative Prompt：

```text
low quality, blurry, watermark, random text, messy background, cheap plastic, oversaturated, distorted shape
```

## 4. 即梦 / 国内图像工具

适合：小红书封面、中文商业海报、快速多风格测试。

建议：中文为主，补充英文材质/镜头关键词。

示例：

```text
液态银色兰花，水银般镀铬花瓣，黑银色高级视觉，柔和轮廓光，大面积留白，适合小红书封面，预留上方标题区，luxury editorial, chrome texture, soft rim light, clean background，避免随机文字、水印、杂乱背景。
```

## 5. Runway / 视频工具

适合：动态海报、品牌KV短视频、开场片头、液态材质动态。

示例：

```text
A liquid silver orchid slowly blooms in a dark luxury studio, mercury chrome petals reflecting soft blue-violet rim light. Slow cinematic push-in, subtle floating particles, elegant mysterious atmosphere, 5 seconds, no text, no watermark, stable subject, smooth motion.
```

## 6. 文字处理建议

- AI生成图中不要依赖复杂中文文字。
- 推荐：先生成“无文字背景图”，再到 Figma / Canva / PPT 中排版。
- Prompt 中写：`reserve a clean title-safe area`，而不是让模型直接生成长标题。
