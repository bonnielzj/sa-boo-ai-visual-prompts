# SA&BOO 2026 AI视觉口令包｜使用说明

本资料包用于 SA&BOO / Bonnie 的文字转图片、视觉风格探索、品牌视觉、室内概念图、小红书封面、提案视觉、产品KV 与 Figma/Canva/MCP 工作流。

## 推荐使用方式

1. 先看 `01_2026视觉趋势转译.md`，选择一个视觉方向。
2. 再用 `02_Prompt操作系统.md` 组织完整口令。
3. 如果需要快速出图，直接复制 `03_可直接复制风格口令库.md` 中的模板。
4. 如果要适配不同平台，看 `04_平台适配指南.md`。
5. 如果要落地到 Figma / Canva / imagegen / PPT，看 `05_MCP与工具路线.md`。

## SA&BOO 核心原则

- 不复制趋势，而是把趋势转译为可控视觉变量。
- 不只写“高级感”，必须写清：材质、光线、构图、留白、镜头、颜色与约束。
- 文字多的画面，建议先生成背景/主体，再在 Figma、Canva 或 PPT 中排版。
- 室内、品牌、商业图都应经过版权、客户隐私、可施工性、商标唯一性检查。


## 视觉资产链表与本地索引

为了避免外部 URL 失效，本资料包已加入本地视觉资产链表：

- `07_视觉资产链表与向量索引.md`
- `visual_assets/visual_asset_linked_list.md`
- `visual_assets/cache/thumbs/`
- `visual_assets/asset_graph.json`
- `scripts/search_visual_assets.py`

示例检索：

```bash
python3 scripts/search_visual_assets.py "液态银 dark chrome"
python3 scripts/search_visual_assets.py "Texture Check 触觉材质"
```

注意：第三方图片缓存仅用于研究索引，不应作为自有商业素材直接使用。
